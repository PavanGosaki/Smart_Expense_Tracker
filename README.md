💰 Smart Expense Tracker

A Python GUI-based application that helps users manage and track their daily expenses efficiently. The application allows users to add, update, delete, search, and analyze expenses with a clean dashboard view.

The project is developed using Python, Tkinter, and SQLite to provide a simple yet powerful desktop expense management system.

🚀 Features
Add new expenses with amount, category, description, and date
Update existing expense records
Delete selected expenses
Search expenses by category
Dashboard showing Total, Today, and Monthly expenses
Export expenses to Excel
Refresh and clear data options
Interactive table view of all records


🛠 Technologies Used
Python
Tkinter (GUI Framework)
SQLite (Database)
Pandas / Excel Export
